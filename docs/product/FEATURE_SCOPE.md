# 功能范围

## MVP 必须交付

| 模块 | 交付内容 |
|---|---|
| 用户 | QQ 登录、资料、退出、个人账本初始化 |
| 个人账本 | 收入、支出、分类、明细、统计 |
| 情侣账本 | 创建、邀请、加入、切换、共同记账、退出、解散 |
| 借贷 | 借入/借出语义、部分处理、结清、逾期、提醒、统计 |
| 周期记账 | 每月/每年/固定期数、自动或确认、暂停、恢复、幂等 |
| 工资 | 工资档案、月度明细、到账、收入联动、工资余额、年度统计 |
| 攒钱目标 | 个人/情侣目标、存入、进度、成员贡献 |
| 通知 | 站内通知和未读状态 |
| 设置 | 主题、默认项、CSV 导出、账号设置 |
| 后台 | 用户、账本关系、任务、通知、反馈和审计查询 |
| 质量 | 权限、幂等、金额、事务、移动适配、主题、测试和部署 |

## 数据边界

- 普通账目通过 `ledger_id` 隔离。
- 工资和个人借贷通过 `user_id` 私有隔离，不自动共享。
- 情侣共同周期规则和共同攒钱目标关联情侣账本。
- 特殊业务生成普通账目时，通过 `source_type + source_id` 建立唯一关联。
- 删除来源记录时，必须明确关联账目是撤销、保留还是一并软删除。

## 交付顺序

1. 文档冻结
2. 工程基础
3. 用户和账本权限
4. 普通记账
5. 借贷
6. 周期记账
7. 工资
8. 攒钱目标
9. 统计和通知
10. 后台、导出和上线质量
