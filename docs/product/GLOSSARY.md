# 术语表

| 术语 | 定义 |
|---|---|
| 个人账本 | 每位用户唯一、仅本人可见的默认账本 |
| 情侣账本 | 最多两名有效成员共同使用的共享账本 |
| 普通账目 | 收入或支出记录 |
| 来源业务 | 工资、借贷处理或周期实例等可生成普通账目的业务记录 |
| 借贷方向 | “我欠别人的”或“别人欠我的” |
| 周期规则 | 定义重复频率和生成方式的模板 |
| 周期实例 | 某条周期规则在某个计划日期的单次执行 |
| 工资月份 | 工资记录归属的自然年月，不等同于到账日期 |
| 工资周期 | 本次发薪到账日至下次预计发薪日前一天 |
| 已存金额 | 初始金额加所有有效存入记录 |
| 整数分 | 以最小货币单位保存金额，如 12.34 元保存为 1234 |
| 幂等 | 相同业务请求或任务重复执行不会产生重复结果 |
| 软删除 | 保留记录但通过删除时间排除正常查询 |
| ExecPlan | 复杂任务的可更新执行计划 |
