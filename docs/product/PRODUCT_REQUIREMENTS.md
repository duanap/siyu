# 产品需求文档

版本：V1.0
产品名称：朝暮同笺 · 四时有余
产品简称：四时有余
品牌规范：`BRAND_IDENTITY.md`
目标终端：手机 Web / PWA，后续可通过 Capacitor 封装安卓应用

## 1. 产品目标

帮助用户低成本管理个人与情侣共同财务，重点解决：

- 日常收入和支出
- 别人欠我、我欠别人的借贷追踪
- 房租、订阅、分期等周期账目
- 工资组成、扣除、年度社保医保公积金明细
- 按发薪周期估算“工资还剩多少”
- 个人或情侣共同攒钱目标

## 2. 用户体系

### 登录

- 支持 QQ OAuth 登录。
- 首次登录创建用户和唯一个人账本。
- 用户资料包括昵称、头像、时区和账号状态。
- 支持退出登录和注销申请。
- 开发环境允许明确标识的本地测试账号；生产环境不得保留测试登录入口。

### 数据归属

- 每位用户有且只有一个有效个人账本。
- 每位用户同一时间最多加入一个有效情侣账本。
- 个人工资、社保、医保、公积金和个人借贷默认仅本人可见。

## 3. 账本

### 个人账本

- 首次登录自动创建。
- 支持个人收入、支出、借贷、周期记账、工资和攒钱目标。
- 只能由本人访问。

### 情侣账本

- 用户可以创建情侣账本并邀请另一位用户加入。
- 最多两名有效成员。
- 双方可新增共同收入和支出。
- 每笔共同账目显示创建人。
- 支持共同周期记账和共同攒钱目标。
- 个人工资和个人借贷不得进入情侣账本统计。
- 支持修改名称、退出、解散和邀请失效处理。

### 账本切换

- 首页、明细和统计支持个人/情侣账本切换。
- 切换后页面数据、筛选和新增记账默认归属同步更新。
- 当前账本选择在当前设备持久化。

## 4. 普通记账

### 类型

- 收入
- 支出

### 字段

- 金额
- 分类
- 业务日期
- 备注（选填）
- 支付方式（选填）
- 图片凭证（选填）
- 账本
- 创建人

### 操作

- 新增、查看、编辑、删除
- 删除需二次确认
- 明细按日期分组
- 支持月份、类型、分类和创建人筛选
- 支持备注关键词搜索
- 情侣账本按权限控制修改和删除

### 分类

系统提供常用收入和支出分类。用户可新增、改名、换图标、排序和停用自定义分类。已被账目使用的分类不得物理删除。

## 5. 借贷

### 类型

- 我欠别人的钱
- 别人欠我的钱

### 借贷记录

字段：

- 对方名称
- 原始金额
- 开始日期
- 到期日期（选填）
- 备注
- 提醒设置
- 当前状态

### 还款与收款

- 支持一次或多次部分还款/收款
- 展示已处理金额和剩余金额
- 处理金额不得超过剩余金额
- 结清后状态自动更新
- 到期未结清时标记逾期
- 用户可选择是否同步生成普通收支账目
- 同一处理记录只能生成一笔关联账目

### 统计

- 当前总负债
- 当前总待收款
- 本月已还
- 本月已收
- 即将到期
- 已逾期

## 6. 周期记账

### 典型场景

- 房租
- 话费、水电燃气
- 会员订阅
- 固定工资
- 保险
- 手机、电脑等分期
- 情侣共同固定支出

### 规则字段

- 名称
- 收入或支出
- 金额
- 分类
- 开始日期
- 重复频率
- 结束日期或总期数（选填）
- 自动生成或到期确认
- 提醒设置
- 归属账本

### 执行方式

- 自动生成：到期时直接生成账目。
- 到期确认：生成待确认实例，用户确认实际金额后入账。
- 每条规则、每个计划日期最多一个执行实例。
- 支持暂停、恢复和结束。
- 分期展示总期数、已完成、剩余期数、每期金额和剩余待付金额。

## 7. 工资

### 工资档案

- 公司或收入来源
- 发薪日
- 默认工资项目模板
- 默认是否到账后生成收入账目
- 支持一个主工资档案；数据结构允许以后增加多个档案

### 月度工资

收入项目：

- 基本工资
- 岗位工资
- 绩效
- 奖金
- 加班费
- 津贴补贴
- 其他收入

扣除项目：

- 养老保险
- 医疗保险
- 失业保险
- 公积金
- 个人所得税
- 其他扣除

系统计算：

- 应发工资
- 个人扣除合计
- 实发工资
- 到账状态和日期

同一工资档案、同一月份只能有一条有效记录。标记到账后可生成一笔金额为实发工资的普通收入，重复操作不得重复生成。

### 工资余额

默认统计周期：

- 本次工资到账日
- 至下一次预计发薪日前一天

展示：

- 本期实发
- 固定支出
- 日常支出
- 已支出合计
- 剩余金额
- 距下次发薪日天数
- 日均可用金额

### 年度明细

按年度展示：

- 应发、实发、奖金
- 养老、医疗、失业保险
- 公积金
- 个税
- 月均实发
- 每月趋势

页面必须提示这些数据来自用户录入，不代表官方账户实时余额。

## 8. 攒钱目标

- 支持个人目标和情侣共同目标
- 目标名称、金额、初始金额、目标日期、封面和备注
- 支持添加、编辑和删除存入记录
- 展示已存、剩余、完成比例和目标日期
- 情侣目标按成员展示贡献金额
- 达到目标金额时自动标记完成
- 完成后保留历史记录

## 9. 首页

### 个人首页

优先展示：

1. 本月收入、支出和结余
2. 工资剩余
3. 当前负债和待收款
4. 下一笔周期账单
5. 攒钱目标
6. 最近账目

### 情侣首页

优先展示：

1. 本月共同收入、支出和结余
2. 我的支出、对方支出
3. 共同周期账单
4. 共同攒钱目标
5. 最近共同账目

## 10. 明细与统计

- 本月、上月、本年和指定月份
- 收入、支出、结余、日均支出和最大单笔支出
- 收支趋势
- 支出分类占比和排行
- 情侣成员支出对比
- 借贷摘要
- 周期账单摘要
- 工资年度摘要

## 11. 通知

站内通知覆盖：

- 情侣邀请、加入、退出和解散
- 借贷到期、逾期
- 周期账单到期、待确认、生成失败
- 工资录入和到账提醒
- 攒钱目标完成

## 12. 设置

- 日间、夜间和跟随系统
- 默认账本
- 默认支付方式
- 金额显示开关
- 通知开关
- CSV 导出
- 账号资料、退出和注销
- 隐私政策、用户协议和版本信息

## 13. 最小运营后台

后台用于运营和故障处理，不直接修改用户财务数据：

- 用户状态管理
- 账本与成员关系查询
- 周期任务执行状态
- 系统通知和公告
- 用户反馈
- 登录与操作审计
- 基础运行数据概览

管理员查看敏感详情必须具备专门权限并产生审计记录。
